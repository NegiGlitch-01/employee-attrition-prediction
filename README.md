# 🏢 Employee Attrition Prediction System

An end-to-end Machine Learning project that predicts whether an employee is likely to leave the company based on workplace, performance, and demographic factors.

The project uses a Support Vector Machine (SVM) model and an interactive Streamlit dashboard to help HR teams identify employees at risk of attrition and take proactive retention measures.

## 🚀 Features

* Employee Attrition Prediction
* Interactive Streamlit Dashboard
* Real-Time Risk Analysis
* Employee Risk Score Visualization
* HR Recommendations Based on Predictions
* Clean and User-Friendly Interface

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Joblib
* Matplotlib

## 📊 Machine Learning Workflow

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Encoding
4. Feature Scaling
5. Model Training (SVM Classifier)
6. Model Evaluation
7. Streamlit Deployment

## ▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Start the application:

streamlit run app.py

## 🎯 Objective

The goal of this project is to help organizations identify employees who may be at risk of leaving, enabling HR teams to improve employee retention and workforce planning.
